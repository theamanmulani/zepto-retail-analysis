# zepto-retail-analysis
This project focuses on analyzing Zepto sales data to generate meaningful business insights using both Excel and Power BI.  The goal is to understand sales performance, identify profit/loss patterns, and create interactive dashboards for better decision-making.

🛠️ Tools & Technologies
Microsoft Excel
Pivot Tables
Charts
Dashboard Design
Power BI
Data Cleaning (Power Query)
Data Modeling
DAX Calculations
Interactive Visualizations
📌 Objectives
Analyze overall sales performance
Track profit and loss trends
Identify top-performing categories/products
Understand customer buying patterns
Build interactive dashboards

📈 Key Insights
Identified high-revenue product categories
Detected loss-making segments
Monthly and category-wise sales trends
Business performance KPIs

📊 Dashboard Features
🔹 Excel Dashboard
Static and quick analysis
Pivot-based insights
Easy-to-understand visuals

🔹 Power BI Dashboard
Interactive filters (slicers)
Dynamic KPIs
Drill-down analysis
Better data visualization
